Синхронизируете этот проект с локальным пустым проектом на компьютере пользователя, через intelij idea (найстройки гита)
После скачивания всех файлов на компьютер, нажимаете кнопку пуск(run)
открываете браузер и заходите на хост http://localhost:8080/
Перед глазами главный экран на котором можно начать делать(Чтобы перейти на страницу админа, введите пароль 12345)
